# 할루시네이션 방지 체크리스트

## 매칭 주장 전 내부 점검 프로토콜 (5-Gate, V3 확장)

### Gate 1: 유물/문헌 실존 확인

매칭에 사용할 유물 또는 고대 문헌에 대해:

- [ ] 박물관명이 실존하는 기관인가?
- [ ] 카탈로그 번호가 구체적으로 알려져 있는가?
- [ ] 발견 연도와 발굴자가 알려져 있는가?
- [ ] 이 유물이 해당 박물관 공식 컬렉션에 등재되어 있는가?

**실패 시 처리**: 카탈로그 번호 없이 "소장처만 명시" 또는 "[번호 확인 필요]" 표기. 번호를 지어내지 않는다.

### Gate 2: 학자/논문 실존 확인

인용할 학술 자료에 대해:

- [ ] 저자가 실존하는 학자인가? (이름이 구체적으로 알려진 인물)
- [ ] 논문/저서 제목이 구체적으로 알려져 있는가?
- [ ] 출판연도와 출판사가 구체적으로 알려져 있는가?
- [ ] 학술지명이 실존하는 학술지인가?

**실패 시 처리**: "학자들에 따르면" 식의 모호한 표현 금지. 구체 인용이 불가능하면 "[구체 출처 확인 필요]" 표기.

### Gate 3: 학계 컨센서스 여부 확인

제시할 주장에 대해:

- [ ] 이것이 주류 학자들의 견해인가, 아니면 소수/비주류 견해인가?
- [ ] 이 동일시(identification)에 학계 논쟁이 있는가?
- [ ] 논쟁이 있다면 주류 측과 소수 측을 균형 있게 제시했는가?

**실패 시 처리**: 논쟁 중인 사항에 "[DEBATED]" 표기. 소수설을 주류설처럼 제시 금지.

### Gate 4: 프로비넌스(provenance) 확인 — V2 추가

매칭에 사용할 유물에 대해:

- [ ] 이 유물이 정식 고고학 발굴(provenanced)에서 출토되었는가?
- [ ] 발굴 책임자·발굴 연도·발굴지 층위가 명확한가?
- [ ] 시장 출현(unprovenanced) 또는 사적 소장만 알려진 유물인 경우, 학계의 위조 평가가 어떻게 되어 있는가?
- [ ] 동일 비문 또는 유사 비문에 대한 위조 보고가 있는가? (예: Goren-Arie 2014 BASOR, Rollston 2003 *Maarav*)

**실패 시 처리**: 시장 출현 + 학계 다수 위조 판정 → `[FORGERY-SUSPECTED]`로 표기, **성경 매칭 증거로 사용 금지**.

대표 위조 의심 사례:
- 베레키야후/바룩 인장 (Goren & Arie 2014, BASOR 372)
- "야고보 납골당"의 후반부 비문 (Lemaire-IAA 위조 재판 결과 무죄지만 학계 진위 논쟁 지속)
- "여호아스 비문" (이스라엘 검찰 위조 판정 → 진위 논쟁)
- 모압어 가짜 비문, 일부 *bullae* 시장 출현물

### Gate 5 (V3 추가): 동명이인·세대 혼동 검사

매칭에 사용할 인물명에 대해:

- [ ] 동일 이름이 다른 시대·다른 가문·다른 직책에 등장하지 않는가?
- [ ] 유물의 인물과 성경의 인물이 *연대*에서 부합하는가? (오차 ± 50년 이내)
- [ ] 유물의 인물과 성경의 인물이 *계보·직책*에서 부합하는가? (왕·아버지·아들·국적 등)
- [ ] 동일 왕조 내 동명 인물(예: 산발랏 I/II/III, 벤하닷 I/II, 헤롯 가문)을 혼동하지 않았는가?
- [ ] 신·구약 동일 이름(예: 마리아·요한·야고보)을 혼동하지 않았는가?

**실패 시 처리**: 동일성 단정 불가 → "동명이인 가능성 있음 — 연대·계보 일치 여부 확인 필요" 명시. 동일성 단정 시 *세대·연대·직책·계보* 일치를 명문화.

대표 동명이인 함정 사례:
- 산발랏 I (느헤미야 시대, 5세기 BC) vs 산발랏 II (WDSP 22 인장, 4세기 BC, I의 손자 가능성)
- 벤하닷 I (왕상 15:18) vs 벤하닷 II (= Adad-idri, 왕상 20장) vs 벤하닷 III (왕하 13장)
- 헤롯 대왕 (마 2장) vs 헤롯 안티파스 (눅 3장 등) vs 헤롯 아그립바 I (행 12장) vs 헤롯 아그립바 II (행 25–26장)
- 사도 야고보 (요한의 형제, 행 12:2 순교) vs 의로운 야고보 (예수의 형제, 행 15장)
- 마리아 — 예수 모친·막달라·베다니·글로바 아내·요한 마가의 어머니 등 다수

### Gate 6 (V3 추가): 폐기된 옛 판독 검사

매칭에 사용할 비문·문헌 판독에 대해:

- [ ] 인용하려는 판독이 19세기·20세기 초 판독인 경우, 후대 학계가 이를 *다수설로 유지*하는가?
- [ ] 후대에 *폐기·재해석*되었다면 옛 판독을 정설인 양 인용하지 않는가?
- [ ] 판독 자체가 학계 논쟁 중이라면 [DEBATED] 표기 후 양설을 균형 있게 제시하는가?

**실패 시 처리**: 폐기된 옛 판독 → 인용 금지, 후대 다수설 우선. 논쟁 중 → [DEBATED] 표기.

대표 폐기·재해석 사례:
- Champollion(1828–29)의 부바스타이트 포털 토포그래픽 명단 "Judah-Melek/Ioudahamalek" 판독 → 현대 학계는 *Yad Hamelek*("왕의 손/기념물") 또는 *Yotbah Hamelek* 등 지명으로 재해석 (Burchardt 1909, Noth 1938, Kitchen 1986, Wilson 2005)
- 메사 비석 31행 "byt[d]wd" 판독: Lemaire 1994 제안 → 2022 RTI 재확인 → Richelle·Burlingame 회의적 → 여전히 [DEBATED]
- "출애굽의 파라오 = 람세스 2세" 단정: 19세기 초 다수설이었으나 현재는 BC 15세기설(투트모세 III)과 BC 13세기설(람세스 II) 양설 미합의

## 절대 금지 목록

| 금지 행동 | 대안 |
|----------|------|
| 없는 카탈로그 번호 창작 | "[번호 미확인]" 또는 알려진 번호만 표기 |
| 없는 학자 이름 창작 | 알려진 학자만 인용, 불확실하면 생략 |
| "대부분의 학자들이..." 모호한 표현 | 구체 학자 이름 + 논문 명시 |
| 소수설을 컨센서스로 제시 | 학계 내 위상 명시 (주류/소수) |
| "증명한다"는 표현 | "일치한다", "지지한다", "확인한다"로 대체 |
| 연대 단정 (논쟁 중인 경우) | "기원전 X세기경", "학계 논쟁 중" 표기 |
| 지명 동일시 단정 | "현재의 X로 비정됨", "동일시 논쟁 중" 표기 |

## 경계선 케이스: 이것은 제시할 수 없음

### 출애굽의 파라오 동일시
- 람세스 2세(기원전 13세기설) vs 투트모세 3세(기원전 15세기설) 논쟁
- 학계 미합의. "출애굽기의 파라오는 람세스 2세" 단정 불가
- 허용 표현: "출애굽 연대는 기원전 13세기 또는 15세기로 학자들 간 논쟁 중이며, 어느 파라오인지 특정되지 않음"

### 노아 방주의 위치
- 고고학·지질학 근거 없음. 제시 불가

### 요나와 역사적 니네베 멸망
- 요나서의 40일 회개 이야기는 역사 기록과 직접 연결 불가. 제시 불가

### 솔로몬 광산 = 팀나 구리 광산
- 상관관계 있으나 "솔로몬의 광산"임을 증명하는 직접 증거 없음. [DEBATED]

## 신뢰도 등급 사용 기준

| 등급 | 조건 | 예시 |
|-----|------|------|
| `[CONFIRMED]` | 물리적 유물 + 학계 주류 합의 | 빌라도 비석, 산헤립 프리즘, 라기스 부조, 쿠르크 모놀리스, 여호야긴 점토판 |
| `[PROBABLE]` | 간접 증거 + 학자 다수 지지 | 고레스 실린더-에스라 연결, 가야바 납골당, 와디 다리예 WDSP 22 |
| `[DEBATED]` | 증거 존재하나 해석 논쟁 | 텔단 비석 일부 해석, 게제르 6실 성문 (고/저년대학) |
| `[UNCERTAIN]` | 증거 불충분, 추론 단계 | 솔로몬 성전 정확한 위치 |
| `[FORGERY-SUSPECTED]` (V2) | 무프로비넌스 + 학계 다수 위조 판정 | 베레키야후/바룩 인장 (Goren-Arie 2014) |
| `[OUT OF SCOPE]` | 역사학·고고학 영역 밖 | 창조, 노아 홍수의 우주적 규모 |
