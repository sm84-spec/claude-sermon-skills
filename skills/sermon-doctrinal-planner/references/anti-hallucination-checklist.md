# 할루시네이션 방지 체크리스트 (Anti-Hallucination Protocol)

본 체크리스트는 sermon-doctrinal-planner의 모든 §1~§8 산출물이 외부로 나가기 전 통과해야 할 4-Gate 검증이다. 단 한 항목이라도 실패하면 해당 인용을 수정하거나 "정확한 인용 직접 확인 권장"으로 대체한다.

---

## Gate 1: 신앙고백·신경 인용의 정확성

§4(역사적 교리 정립) 및 §6(설교 구조) 등에서 신앙고백·신경을 인용할 때:

- [ ] 인용한 신앙고백의 이름·연도가 정확한가?
  - 사도신경 (Apostles' Creed, 8세기 텍스투스 레켑투스)
  - 니케아-콘스탄티노폴리스 신경 (AD 381) — "니케아 신경"은 통상 381년 본문
  - 칼케돈 신앙정의 (AD 451)
  - 아타나시우스 신경 (*Quicumque Vult*, c. 5-6세기 라틴 서방)
  - 웨스트민스터 신앙고백 (1647), 대요리문답 (1648), 소요리문답 (1647)
  - 하이델베르크 요리문답 (1563)
  - 도르트 신경 (1618-19)
  - 벨기에 신앙고백 (1561)
  - 한국 장로교 12신조 (1907)

- [ ] 조항 번호·문답 번호가 정확한가? (`creeds-and-confessions.md`에 수록된 번호와 대조)

- [ ] 인용된 본문이 표준 한국어 번역본과 일치하는가?

- [ ] 본 references/creeds-and-confessions.md에 수록되지 않은 조항을 인용하려 하는가?
  - **YES** → "정확한 본문은 직접 확인 권장"이라고 명시. 인용 문구를 만들어내지 않는다.
  - **NO** → 수록된 인용 그대로 사용.

**금지**:
- WCF 5.7 (실제 5장은 7항까지 있음)이라고 표기하면서 다른 조항의 내용을 가져다 쓰기
- "하이델베르크 요리문답 21문답"이라고 인용하면서 다른 문답의 내용을 쓰기
- 도르트 신경에 없는 "TULIP" 용어를 도르트 신경 본문 인용처럼 표기하기

## Gate 2: 성경 본문 인용의 정확성

§2(본문 분석)·§3(연관 보조 본문)에서 성경 인용 시:

- [ ] 장절 표기가 정확한가? (예: 엡 2:8-10, 롬 5:1, 빌 2:5-11)
- [ ] 인용된 본문이 개역개정 또는 표준 한국어 번역과 실제로 일치하는가?
- [ ] 보조 본문이 정말 §1의 교리를 가르치는가? (단순히 비슷한 단어만 있고 다른 맥락은 아닌가)
- [ ] 원어(헬라어·히브리어) 인용 시:
  - [ ] 단어 형태가 실제 본문에 등장하는 형태인가?
  - [ ] 음역(transliteration)이 표준적인가?
  - [ ] 의미가 BDAG·HALOT 등 표준 사전과 일치하는가?
  - [ ] 모르는 단어를 만들어내고 있지 않은가?

**원어 인용 안전 규칙**:
- 확실히 아는 핵심 단어(예: *dikaiosynē* 의, *charis* 은혜, *pistis* 믿음, *sōtēria* 구원, *euangelion* 복음, *agapē* 사랑, *homoousios* 동일본질, *hypostasis* 위격, *prosōpon* 인격)만 사용
- 본문에 등장하지 않는 단어를 "본문이 사용한 단어"라고 표기 금지
- 의미가 확실하지 않으면 한국어 의미만 제시하고 원어 형태 표기 보류

### 인용 검증 체크

- [ ] 인용하려는 저자가 실존 인물인가?
- [ ] 인용하려는 저작이 실제로 그 저자가 쓴 것인가? (오인 위험: 아타나시우스 신경은 아타나시우스 본인 저작 아님)
- [ ] 출판연도/원작 연도가 정확한가?
- [ ] 저작 제목의 한국어 번역이 일반적인가?
- [ ] 페이지 번호·장 번호를 임의로 만들고 있지 않은가?

**금지**:
- 실제 인용 페이지·장을 모르면서 "Calvin, *Institutes* 3.11.1"처럼 정확한 위치 표기
- 칼빈이 쓴 적 없는 책 제목을 칼빈 저작처럼 표기
- 어거스틴이 한 적 없는 말을 어거스틴 인용으로 표기

**안전 규칙**: 정확한 위치(페이지·장·절)를 모르면 "(예: 칼빈 『기독교 강요』 3권 칭의론 부분)"처럼 일반적 위치만 표기.

## Gate 4: 본문 → 교리 추론의 타당성

§3(교리적 함의 도출)에서 본문이 §1 교리 명제로 수렴된다고 주장할 때:

- [ ] 본문이 §1 교리 명제를 **직접** 가르치는가, 아니면 **간접적으로 함의**할 뿐인가?
- [ ] 본문의 1차 의미(*sensus literalis*)가 그 교리와 일치하는가, 아니면 우의적·도덕적 적용일 뿐인가?
- [ ] 본문의 문맥(직전·직후·책 전체·언약 흐름)이 그 교리 진술과 일치하는가?
- [ ] 사용자가 제시한 본문이 정말 그 교리설교에 적합한가? (적합하지 않다면 더 나은 본문을 제안)

**금지**:
- 본문에 없는 단어·개념을 본문이 가르친다고 진술
- 다른 본문의 의미를 현 본문에 끌어들여 "본문이 가르친다"고 진술 (proof-texting 오류)
- 본문의 1차 의미를 무시하고 알레고리로 교리를 끌어내기

**경계선 사례 처리**:
- 시편 한 절 외침 (예: "내 영혼아 잠잠히 하나님만 바라라" 시 62:5) → 강해·묵상에 적합, 한 개의 교리설교 본문으로는 좁다 → sermon-qt-original-text-based 또는 sermon-calvin-style-insight로 안내
- 잠언 단일 구절 → 지혜문학 특성상 단일 교리설교 본문보다는 잠언 한 단락 또는 주제별 묶음 권유
- 묵시문학 본문 (단·계) → 종말론 분과로 한정하되, 표상의 다의성에 주의

## Gate 5 (보조): 학계 주류 입장 확인

§4·§5에서 특정 신학적 입장이 "학계 주류" 또는 "정통"이라고 진술할 때:

- [ ] 그 입장이 정말 개혁주의 정통 또는 보편 교회 정통인가?
- [ ] 소수설을 주류설처럼 진술하고 있지 않은가?
- [ ] 정통 내부의 의견 차이를 단일 정답처럼 다루고 있지 않은가? (예: 천년왕국 입장 — 무천년·후천년·역사적 전천년 모두 개혁주의 정통 내부 의견 차이)

**금지**:
- "모든 개혁주의자는 X라고 가르친다" — 실제로는 의견이 갈리는 경우
- 18-19세기 자유주의 신학자의 입장을 정통 학계 주류로 제시
- 한 신학자의 사견을 "교회의 가르침"으로 표기

| 상황 | 처리 |

| Gate 2 실패 (본문이 그 교리를 직접 가르치지 않음) | 더 적합한 본문 제안 또는 보조 본문으로 강등 |

| Gate 5 실패 (소수설을 주류로) | "이는 [Aiu] 의견이다" 또는 "정통 내부에서 [X] 입장과 [Y] 입장이 함께 받아들여진다" 명시 |
| 두 개 이상 Gate 실패 | 해당 인용·진술 전체 삭제, 일반적 방향만 제시 |

## 산출 직전 최종 자기 점검 (8-Gate 확장)

1. 이 기획안의 §4에 인용된 모든 신앙고백 조항이 `creeds-and-confessions.md`에 수록되어 있는가? 그렇지 않다면 "(직접 확인 권장)"이라 표기했는가?
2. §2의 원어 인용 단어들이 모두 실제 본문에 등장하는 단어인가? BDAG·HALOT 등 표준 사전 의미와 일치하는가?
3. §3의 본문 → 교리 추론 흐름에 비약이나 proof-texting이 없는가? 본문의 1차 의미(*sensus literalis*)가 §1 명제를 직접 지지하는가?
4. §8의 추천 도서 중 저자·제목·연도를 모두 명확히 아는 것만 추천했는가? 모호한 것은 "(직접 확인 권장)" 표기했는가?
5. 어디에도 "○○에 따르면" 식 모호 인용이 없는가?
6. **(추가 6)** 정통 내부 의견 차이를 단일 정답처럼 다루고 있지 않은가? (천년왕국·창조 연대·은사·세례 대상 등) — 한국 장로교 합동 기본 입장 + 다른 정통 입장 명시했는가?
7. **(추가 7)** 한국 이단·단체명을 사용할 때 SKILL.md "단체명 사용 원칙"을 준수했는가? (사실 검증·정통 진술·인격 비방 회피)
8. **(추가 8)** 자주 도전받는 본문(`text-to-doctrine-mapping.md` Case 7)이 본문일 경우, 도전 해석과 개혁주의 표준 해석을 균형 있게 §5에 진술했는가?

위 8개 점검을 모두 통과한 기획안만 산출.
