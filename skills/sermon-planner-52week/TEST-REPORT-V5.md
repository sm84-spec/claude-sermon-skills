## 결정적 검증으로 차단된 할루시네이션 사례

V5 검증 과정에서 LLM의 *임의 생성 위험*을 Python 도구가 차단한 패턴들:

1. **2028년 53주 케이스** — 종전 LLM은 12월 31일을 "52주차"로 강제 매핑할 위험. easter_calculator.py가 53주 사실을 명시적으로 보고하고 SKILL.md "53주 케이스" 규약을 적용하도록 안내.
2. **2035년 19주차 성령강림+어버이 충돌** — LLM이 임의로 한 절기를 다른 주차로 옮길 위험. compute_all_week_numbers가 충돌을 명시적으로 보고.
3. **2031년 추수감사 = 47주? 46주?** — LLM이 일반 안내("47주차")를 무비판 적용할 위험. 자동 산출은 정확히 46주차 반환, V5 보고서는 이 차이를 명시.
4. **풀 외 찬송가 등재 시도** — hymn_validator가 모든 (번호, 제목) 쌍을 검증, 풀 밖이면 즉시 거부.
5. **정경 범위 밖 본문 인용 시도** — bible_validator가 모든 책·장·절 범위 검증.
