### # 1. 시 121편 표준 QT

**위험 지점**: 시 121:1 "올꼬"(KRV) vs "올까"(GAE), 121:3 "너로 실족지 않게"(KRV) vs "너를 실족하지 아니하게"(GAE), "지키시는 자가"(KRV) vs "지키시는 이가"(GAE).

**도구 1차 실행 결과**:
```
$ python3 scripts/verify_quote.py \
    --text "내가 산을 향하여 눈을 들리라 나의 도움이 어디서 올꼬 ..."
[WARN] KRV 흔적 4건 검출:
  - 시 121:1: KRV '어디서 올꼬' → GAE '어디서 올까'
  - 시 121:3: KRV '너로 실족지 않게' → GAE '너를 실족하지 아니하게'
  - 시 121:3: KRV '지키시는 자가' → GAE '지키시는 이가'
```

**도구 검출 후 재작성 → 2차 실행**: KRV 흔적 0건. 본문 인용 정확.

**핵심 원어 활용**:
- שָׁמַר (šāmar, 샤마르) — "지키다" — 본문에 반복 6회 (3·4·5·7·8절)
- עֵזֶר (ʿēzer, 에제르) — "도움" (2절)
- 대지3은 **기본형** (한 단어 שָׁמַר의 세 의미 층 — 영혼·일상·종말)

**C1-C8 + Python**: ALL PASS (재작성 1회 후).
