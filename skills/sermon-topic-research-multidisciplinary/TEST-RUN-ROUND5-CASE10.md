# 라운드 5 케이스 10 — 도시화와 향수 (urbanization & nostalgia)

**입력 프롬프트**: "도시화·이주·고향 상실·향수 주제로 학제간 설교 자료."

표준 양식으로 산출했습니다. 더 간단히 또는 더 상세히 원하시면 말씀해 주십시오.

## 0. 주제 개요 (Overview)

- **주제어**: 도시화와 향수 (고향 상실·뿌리)
- **본 조사의 범위**: 11개 분야에서 한국 도시화·이주·향수의 결.
- **주제의 한국적 시의성**: 한국은 수도권 인구 집중·지방 소멸이 동시에 진행되며 회중 다수가 고향을 떠나 도시에 산다.
- **이 조사로 설교자가 얻는 것**: "뿌리"와 "나그네"의 신학을 한국 회중 현실에 닿게 하는 컨텍스트.

## 1. 사회

**핵심 진단**: 한국 수도권 집중·지방 소멸이 동시에 진행 중이다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 수도권 인구 비중은 통계청 인구주택총조사 직접 확인 권장 (지속 증가 추세)
- 이론·개념: 도시화 이론 — Henri Lefebvre *La production de l'espace* (Anthropos 1974) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 지방소멸 위기 — 행정안전부·한국고용정보원 직접 확인 권장

**현재 흐름과 미래 전망**: 수도권 집중 흐름 지속.

**설교 활용 포인트**: 회중의 이주·뿌리 상실을 강단이 다룬다.

## 2. 기술

**핵심 진단**: 디지털 통신이 물리적 거리를 줄였지만 향수를 해소하지는 못한다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 가족 영상통화 사용은 한국정보통신정책연구원(KISDI) 직접 확인 권장
- 이론·개념: 디지털 친밀성(digital intimacy)
- 사례·사건: 명절 영상통화 보편화

**현재 흐름과 미래 전망**: VR·AR이 향수 경험을 변형 가능.

**설교 활용 포인트**: 기술이 답하지 못하는 향수의 자리를 신학적으로 짚는다.

## 3. 산업

**핵심 진단**: 한국 부동산·교통·물류 산업이 도시화의 산업적 토대다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 수도권 부동산 가격은 한국부동산원 직접 확인 권장
- 이론·개념: 도시 경제학
- 사례·사건: 한국 GTX·SRT 등 교통 인프라 확장 — 국토교통부 직접 확인 권장

**현재 흐름과 미래 전망**: 수도권 집중 산업 구조 지속.

**설교 활용 포인트**: 회중의 "왜 도시에 사는가"를 신학적으로 묻는다.

## 4. 경제

**핵심 진단**: 도시 임금·주거비·교육비가 이주의 경제적 동인이자 부담이다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 수도권 주거 부담 지수(PIR)는 KB금융지주·한국부동산원 직접 확인 권장
- 이론·개념: 도시 임금 프리미엄(urban wage premium)
- 사례·사건: 한국 청년 수도권 이주 — 통계청 직접 확인 권장

**현재 흐름과 미래 전망**: 청년 이주·주거 부담 지속.

**설교 활용 포인트**: 회중의 경제적 이주를 신학적으로 위로하고 분별.

## 5. 환경

**핵심 진단**: 도시 환경(녹지·소음·미세먼지)이 회중의 일상에 영향을 준다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 도시 녹지율은 환경부·국토교통부 직접 확인 권장
- 이론·개념: 바이오필리아(biophilia) — Edward O. Wilson *Biophilia* (Harvard University Press 1984) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 도시공원·하천 복원 — 환경부 직접 확인 권장

**현재 흐름과 미래 전망**: 도시 녹지·생태 회복 운동.

**설교 활용 포인트**: 도시 안 창조세계의 흔적을 회중과 함께 본다.

## 6. 정치

**핵심 진단**: 수도권·지방 정책 균형이 한국 정치의 핵심 의제다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 국가균형발전 정책은 국가균형발전위원회 직접 확인 권장
- 이론·개념: 균형발전·분권 이론
- 사례·사건: 한국 행정수도 이전 논의 — 국토교통부 직접 확인 권장. 특정 정당 옹호·비판 금지.

**현재 흐름과 미래 전망**: 분권·균형 정책 논의 지속.

**설교 활용 포인트**: 회중의 지역 책임을 신학적으로 깨운다.

## 7. 국제관계

**핵심 진단**: 글로벌 도시화·이주 흐름이 한국 도시에도 영향을 준다.

**주요 데이터·이론·사상가**:
- 데이터 1: UN World Urbanization Prospects 보고서 — 직접 확인 권장
- 이론·개념: 글로벌 도시(global city) — Saskia Sassen *The Global City* (Princeton 1991) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 서울의 글로벌 도시 지위 — UN-Habitat 직접 확인 권장

**현재 흐름과 미래 전망**: 글로벌 도시 네트워크 강화.

**설교 활용 포인트**: 회중의 도시적 시민권을 신학적으로 인식.

## 8. 법

**핵심 진단**: 한국 국토기본법·도시계획법·국가균형발전특별법이 핵심 법제다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 국가균형발전특별법 제정 2004 — 법제처 직접 확인 권장
- 이론·개념: 공간 정의(spatial justice)
- 사례·사건: 한국 지방분권 입법 — 직접 확인 권장

**현재 흐름과 미래 전망**: 지방분권 법제 강화.

**설교 활용 포인트**: 회중의 공간적 권리·책임을 신학적으로 풀어낸다.

## 9. 제도

**핵심 진단**: 한국 도시·농촌 교회·디아스포라 한인 교회의 협력 망이 있다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 농어촌 교회 통계는 한국기독교사회문제연구원·각 교단 직접 확인 권장
- 이론·개념: 도시 선교(urban mission)
- 사례·사건: 한국 농촌 교회·도시 교회 결연 — 직접 확인 권장

**현재 흐름과 미래 전망**: 디아스포라·도시 교회 협력 확장.

**설교 활용 포인트**: 회중의 뿌리와 현주소를 동시에 안아낸다.

## 10. 심리

**핵심 진단**: 향수(nostalgia)·소속감 상실·정체성 혼란이 핵심 심리 메커니즘이다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 도시 거주자 정신건강·외로움 통계는 보건복지부·통계청 직접 확인 권장
- 이론·개념: Constantine Sedikides 등의 향수 심리학 연구 (현대 심리학 학술지)
- 사례·사건: 한국 1인 가구 외로움 — 통계청 직접 확인 권장

**현재 흐름과 미래 전망**: 향수의 심리적 회복 기능 연구 확장.

**설교 활용 포인트**: 향수의 신학적 의미(영원한 본향 갈망)를 회중에 전한다.

## 11. 영성

**핵심 진단**: 나그네 신학·하늘의 본향·언약 백성의 정체성이 신학적 핵심이다.

**주요 데이터·이론·사상가**:
- 데이터 1: 한국 디아스포라 교회 통계는 한국세계선교협의회(KWMA) 직접 확인 권장
- 이론·개념: 어거스틴 De Civitate Dei (라틴 원작 413–426) — 두 도성과 천상의 시민권 / 칼빈 『기독교 강요』 (1559 라틴 최종판) 3권 — 그리스도인의 삶과 본향 묵상 / 박윤선 『성경주석』 (영음사 1953–1979 시리즈) 히브리서편
- 사례·사건: 한국 디아스포라 한인 교회의 정체성 신학

**현재 흐름과 미래 전망**: 나그네 신학의 한국적 토착화.

**설교 활용 포인트**: 회중에게 영원한 본향의 신학을 전한다.

## 12. 종합 인사이트 (Synthesis)

- **관통 키워드**: 이주·뿌리·향수·본향·시민권·언약
- **분야 간 충돌·긴장**: 도시 기회(§4) vs. 뿌리 상실(§10·§11). / 글로벌 도시(§7) vs. 지방 소멸(§1·§6).
- **회중이 처한 현실**: 회중 다수가 고향을 떠나 도시에 산다. 강단의 한 마디가 뿌리와 본향의 신학을 회복시킬 수 있다.
- **본문이 답해야 할 질문**: 그리스도인은 어떤 시민인가? 본향은 어디인가? 뿌리와 본향의 관계는?
