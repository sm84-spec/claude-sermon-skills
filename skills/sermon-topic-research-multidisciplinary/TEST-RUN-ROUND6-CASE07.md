# 라운드 6 케이스 7 — 배신 (betrayal)

**입력 프롬프트**: "배신 주제로 학제간 설교 자료."

표준 양식으로 산출했습니다. 더 간단히 또는 더 상세히 원하시면 말씀해 주십시오.

## 0. 주제 개요 (Overview)
- **주제어**: 배신
- **본 조사의 범위**: 11개 분야에서 배신 경험·회복 분석.
- **주제의 한국적 시의성**: 한국 사회 신뢰 위기 가운데 회중도 배신 경험을 안고 있다.
- **이 조사로 설교자가 얻는 것**: 화해·용서·정의의 신학 컨텍스트.

## 1. 사회
**핵심 진단**: 한국 사회 신뢰 위기가 배신 경험을 증폭한다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 사회 신뢰 지수는 한국행정연구원 사회통합실태조사 직접 확인 권장
- 이론·개념: Anthony Giddens *The Consequences of Modernity* (Stanford 1990) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 보이스피싱·전세사기 보고
**현재 흐름과 미래 전망**: 신뢰 회복이 사회 과제.
**설교 활용 포인트**: 회중의 신뢰 상처를 강단이 안는다.

## 2. 기술
**핵심 진단**: 디지털 사기·딥페이크가 새로운 배신을 만든다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 디지털 사기 신고는 경찰청 직접 확인 권장
- 이론·개념: 디지털 신뢰 (digital trust)
- 사례·사건: 한국 딥페이크 범죄
**현재 흐름과 미래 전망**: AI 사기 증가.
**설교 활용 포인트**: 디지털 시대 신뢰 분별.

## 3. 산업
**핵심 진단**: 한국 보이스피싱·전세사기 등 사기 산업이 변질되었다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 보이스피싱 피해액은 경찰청·금융감독원 직접 확인 권장
- 이론·개념: 신뢰의 산업적 가치
- 사례·사건: 한국 금융 사기 — 금융감독원 직접 확인 권장
**현재 흐름과 미래 전망**: 사기 진화.
**설교 활용 포인트**: 회중을 향한 분별 사역.

## 4. 경제
**핵심 진단**: 한국 가계 사기 피해가 누적된다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 금융 사기 피해액은 금융감독원 직접 확인 권장
- 이론·개념: 신뢰의 경제 — Putnam·Fukuyama
- 사례·사건: 한국 전세사기 — 국토교통부 직접 확인 권장
**현재 흐름과 미래 전망**: 사기 피해 지속.
**설교 활용 포인트**: 회중에 위로와 신학적 회복.

## 5. 환경
**핵심 진단**: 환경 약속 위반(그린워싱)도 일종의 배신이다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 ESG 그린워싱 사례는 환경부 직접 확인 권장
- 이론·개념: 그린워싱
- 사례·사건: 한국 기업 환경 약속 미이행
**현재 흐름과 미래 전망**: 그린워싱 규제 강화.
**설교 활용 포인트**: 약속의 신학.

## 6. 정치
**핵심 진단**: 한국 정치 약속 불이행이 회중의 정치 신뢰를 무너뜨린다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 정치 신뢰 지수는 한국갤럽 직접 확인 권장
- 이론·개념: 정치적 약속·공약
- 사례·사건: 한국 공약 이행 평가. 특정 정당 옹호·비판 금지.
**현재 흐름과 미래 전망**: 정치 신뢰 회복 과제.
**설교 활용 포인트**: 회중의 정치 분별.

## 7. 국제관계
**핵심 진단**: 국가 간 협약 위반이 국제 신뢰를 무너뜨린다.
**주요 데이터·이론·사상가**:
- 데이터 1: 국제 협약 이행 평가는 유엔·OECD 직접 확인 권장
- 이론·개념: 국제 신뢰
- 사례·사건: 한국 외교 약속 — 외교부 직접 확인 권장
**현재 흐름과 미래 전망**: 국제 신뢰 회복.
**설교 활용 포인트**: 글로벌 약속의 신학.

## 8. 법
**핵심 진단**: 한국 형법 사기·배임·횡령죄가 배신의 법적 양상이다.
**주요 데이터·이론·사상가**:
- 데이터 1: 형법 제347조(사기)·제355조(횡령·배임) — 법제처 직접 확인 권장
- 이론·개념: 신의성실 원칙(민법 제2조) — 법제처 직접 확인 권장
- 사례·사건: 한국 배임 판례 — 대법원 직접 확인 권장
**현재 흐름과 미래 전망**: 디지털 사기 입법.
**설교 활용 포인트**: 법과 회복의 신학.

## 9. 제도
**핵심 진단**: 한국 회복적 정의 제도가 도입되고 있다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 회복적 사법 시범은 법무부 직접 확인 권장
- 이론·개념: 회복적 정의(restorative justice) — Howard Zehr *Changing Lenses* (Herald Press 1990) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 회복적 정의 모델 — 직접 확인 권장
**현재 흐름과 미래 전망**: 회복적 정의 확산.
**설교 활용 포인트**: 회복의 신학과 제도.

## 10. 심리
**핵심 진단**: 배신 트라우마·신뢰 상실·복수 충동이 핵심 심리다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 배신 트라우마 연구는 한국심리학회지 직접 확인 권장
- 이론·개념: Jennifer Freyd 배신 외상 이론(*Betrayal Trauma*, Harvard 1996) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 PTSD 사례
**현재 흐름과 미래 전망**: 배신 트라우마 회복 모델 확산.
**설교 활용 포인트**: 신학적 회복의 자리.

## 11. 영성
**핵심 진단**: 유다의 배신·베드로의 부인·화해의 신학이 핵심이다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 교회 회복 사역은 한국기독교사회문제연구원 직접 확인 권장
- 이론·개념: 어거스틴 De Civitate Dei (라틴 원작 413–426) — 두 도성과 배신 / 칼빈 『기독교 강요』 Institutio (1559 라틴 최종판) — 회개와 회복 / Miroslav Volf *Exclusion and Embrace* (Abingdon 1996) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 교회 회복 운동
**현재 흐름과 미래 전망**: 회복 신학의 한국적 토착화.
**설교 활용 포인트**: 회중의 배신 경험을 회복의 자리로.

## 12. 종합 인사이트 (Synthesis)
- **관통 키워드**: 신뢰·약속·배신·트라우마·회개·회복
- **분야 간 충돌·긴장**: 정의(§8) vs. 용서(§11). / 분노·복수(§10) vs. 화해(§11).
- **회중이 처한 현실**: 회중 가운데 배신 경험자가 있다.
- **본문이 답해야 할 질문**: 배신 앞의 신학적 응답은? 용서와 정의의 관계는? 베드로의 회복은?
