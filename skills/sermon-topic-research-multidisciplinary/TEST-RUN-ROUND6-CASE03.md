# 라운드 6 케이스 3 — 노화 (aging)

**입력 프롬프트**: "노화·고령화 주제로 학제간 설교 자료."

표준 양식으로 산출했습니다. 더 간단히 또는 더 상세히 원하시면 말씀해 주십시오.

## 0. 주제 개요 (Overview)
- **주제어**: 노화
- **본 조사의 범위**: 11개 분야에서 한국 노화·고령화의 다층 분석.
- **주제의 한국적 시의성**: 한국 합계출산율 0.72(통계청 2023) 시대에 초고령화가 가속화된다.
- **이 조사로 설교자가 얻는 것**: 노년의 신학을 한국 현실에 닿게 하는 컨텍스트.

## 1. 사회
**핵심 진단**: 한국 초고령사회 진입이 임박했다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 고령인구 비율은 통계청 장래인구추계 직접 확인 권장
- 이론·개념: 활동이론(activity theory)·이탈이론(disengagement theory) — Cumming·Henry 1961
- 사례·사건: 한국 노인 단독가구 증가
**현재 흐름과 미래 전망**: 초고령사회 진입.
**설교 활용 포인트**: 회중의 노년이 강단의 자리.

## 2. 기술
**핵심 진단**: 디지털 격차가 노인층에 가장 크다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 노인 디지털 정보화 수준은 한국지능정보사회진흥원(NIA) 직접 확인 권장
- 이론·개념: 디지털 격차(digital divide)
- 사례·사건: 한국 노인 무인 키오스크 접근성 — 직접 확인 권장
**현재 흐름과 미래 전망**: AI 돌봄 로봇 등장.
**설교 활용 포인트**: 회중의 노인 디지털 약자 보호.

## 3. 산업
**핵심 진단**: 한국 실버산업·돌봄 산업이 빠르게 성장한다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 노인복지시설 수는 보건복지부 직접 확인 권장
- 이론·개념: 시니어 경제(silver economy)
- 사례·사건: 한국 장기요양보험 시행 2008-07-01
**현재 흐름과 미래 전망**: 돌봄 산업 확장.
**설교 활용 포인트**: 청지기 돌봄의 신학.

## 4. 경제
**핵심 진단**: 한국 노인빈곤율이 OECD 최고 수준이다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 노인빈곤율은 통계청·OECD 직접 확인 권장 (OECD 비교 최고 수준)
- 이론·개념: 연금 체계 비교
- 사례·사건: 한국 국민연금 도입 1988
**현재 흐름과 미래 전망**: 연금 개혁 논의.
**설교 활용 포인트**: 빈곤한 노인을 향한 공동체 책임.

## 5. 환경
**핵심 진단**: 노인은 환경 위험에 더 취약하다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 폭염 사망자 중 노인 비율은 보건복지부·질병관리청 직접 확인 권장
- 이론·개념: 환경 정의 — 약자 노출
- 사례·사건: 한국 쪽방촌 노인 폭염 사망
**현재 흐름과 미래 전망**: 기후 변동 노인 영향.
**설교 활용 포인트**: 약자 보호의 신학.

## 6. 정치
**핵심 진단**: 한국 노인 복지 정책이 정치 의제다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 기초연금 시행 2014-07-01 — 보건복지부 직접 확인 권장
- 이론·개념: 세대 정치
- 사례·사건: 한국 노인 정책 변동. 특정 정당 옹호·비판 금지.
**현재 흐름과 미래 전망**: 노인 복지 확대 논의.
**설교 활용 포인트**: 회중의 노인 정책 분별.

## 7. 국제관계
**핵심 진단**: 한국 고령화 속도는 OECD 최고 수준이다.
**주요 데이터·이론·사상가**:
- 데이터 1: OECD 고령화 비교는 OECD 직접 확인 권장
- 이론·개념: 인구 변동 이론
- 사례·사건: 일본 초고령사회 진입 사례
**현재 흐름과 미래 전망**: 동아시아 고령화 가속.
**설교 활용 포인트**: 글로벌 노화 안에서 회중의 자리.

## 8. 법
**핵심 진단**: 한국 노인복지법·장기요양보험법이 핵심이다.
**주요 데이터·이론·사상가**:
- 데이터 1: 노인복지법 제정 1981 — 법제처 직접 확인 권장
- 이론·개념: 노인 인권
- 사례·사건: 한국 노인학대 신고 — 보건복지부 직접 확인 권장
**현재 흐름과 미래 전망**: 노인 인권 법제 강화.
**설교 활용 포인트**: 법과 신학의 만남.

## 9. 제도
**핵심 진단**: 한국 노인복지관·요양시설·교회 노인사역이 협력한다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 노인종합복지관 수는 보건복지부 직접 확인 권장
- 이론·개념: 노인 돌봄 통합 모델
- 사례·사건: 한국 교회 노인대학 운영
**현재 흐름과 미래 전망**: 교회·지역 협력 확장.
**설교 활용 포인트**: 교회가 노년의 자리가 되는 신학.

## 10. 심리
**핵심 진단**: 노년기 우울·인지 저하·죽음 불안이 핵심 심리다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 노인 우울 유병률은 보건복지부 노인실태조사 직접 확인 권장
- 이론·개념: Erik Erikson 8단계 발달이론 — 노년기 자아통합 vs. 절망 (*Childhood and Society* Norton 1950 / *The Life Cycle Completed* Norton 1982) — 한국어 출판 정보 직접 확인 권장
- 사례·사건: 한국 노인 자살률 증가
**현재 흐름과 미래 전망**: 노년 정신건강 회복 프로그램.
**설교 활용 포인트**: 노년의 자아통합을 신학적으로 풀어낸다.

## 11. 영성
**핵심 진단**: 노년기 영성·죽음 묵상·신앙 유산이 신학적 핵심이다.
**주요 데이터·이론·사상가**:
- 데이터 1: 한국 교회 노년기 사역은 한국기독교사회문제연구원 직접 확인 권장
- 이론·개념: 어거스틴 De Civitate Dei (라틴 원작 413–426) — 두 도성과 종말 / 칼빈 『기독교 강요』 Institutio (1559 라틴 최종판) 3권 — 죽음 묵상(meditatio mortis) / 박형룡 『교의신학』 (한국개혁주의신행협회 1964–1979 시리즈) 종말론편
- 사례·사건: 한국 권사·장로 세대의 신앙 유산
**현재 흐름과 미래 전망**: 노년 영성 회복 흐름.
**설교 활용 포인트**: 회중의 노년을 신앙 유산의 자리로.

## 12. 종합 인사이트 (Synthesis)
- **관통 키워드**: 고령화·빈곤·돌봄·유산·죽음 묵상·자아통합
- **분야 간 충돌·긴장**: 경제 효율(§3·§4) vs. 노인 돌봄(§9·§11). / 세대 갈등(§6) vs. 신앙 유산(§11).
- **회중이 처한 현실**: 회중 다수가 노년을 향해 간다. 강단이 노년의 자리를 마련한다.
- **본문이 답해야 할 질문**: 노년이란 무엇인가? 신앙 유산은? 죽음 묵상은?
