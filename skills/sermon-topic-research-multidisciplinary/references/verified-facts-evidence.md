## E7. 검증 사이클 추가 절차 — 영구 편입

본 검증을 거치며 발견된 결함이 보여주듯, 결정론 검증기 외에 *외부 1차 출처 직접 대조*가 반드시 필요하다. 이를 영구 절차로 편입:

1. 산출물 작성 → `scripts/validate_output.py` 결정론 검증 → critical 0건 확인
2. **추가 단계**: 산출물에 인용된 *모든 정확 수치·출판년도·법령 시행일*이 `references/verified-facts.md` 등재 항목인지 확인
3. **추가 단계**: 신규 사실은 외부 1차 출처(법령정보센터·통계청 보도자료·UNFCCC 등) 직접 조회 후 `verified-facts.md`에 추가 + 본 `verified-facts-evidence.md`에 증거 단위(En) 기록
4. 출처 없는 사실은 인용 *금지* — "직접 확인 권장"으로 처리 (G1·G9)

이 절차를 SKILL.md "결정론 환원" 절에 반영한다.
