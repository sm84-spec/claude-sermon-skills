# 검증된 사실 — 1:1 외부 출처 대조표

본 스킬이 통계·연도·도서 정보를 인용할 때 *결정론으로 환원 불가능한* 사실은 본 파일의 *외부 출처 1:1 대조*를 통해서만 검증한다. 본 파일에 없는 사실은 G1·G2·G4·G9에 따라 "직접 확인 권장"으로 표기한다.

## 5. 본 파일에 없는 정보 처리 절차

1. 인용하려는 통계·저작·본문이 본 파일에 *없다* → 본문에 인용하지 *않는다*.
2. 일반 경향만 진술하고 "정확한 자료는 [기관] 직접 확인 권장" 명시.
3. *반드시* 본 파일에 등재된 항목만 정확 수치·연도 표기.
4. 새로 검증된 사실은 본 파일에 *먼저 추가*한 뒤 인용한다.

**규약**: 본 파일에 없는 정확한 수치·연도를 인용하면 G1·G2·G4 위반.
