### V3-2 발견: 위경 책명 prefix 거짓 양성

**문제**: "4 에스라 13장" (위경 4 Ezra, Pseudepigrapha)이 Python passage 검증에서 *구약 에스라 13장*으로 매칭되어 CHAPTER_OUT_OF_RANGE 거짓 양성 발생.

**보강**: `citation_verifier.py`에 위경 prefix 인식 로직 추가 — 책명 직전 *4 이상의 정수*가 있으면 *위경/외경*으로 간주하고 검증 건너뜀. 정경 prefix는 1·2·3만 (1 사무엘, 2 베드로, 3 요한 등).

```python
# 위경/외경 prefix 인식
start = m.start("book")
if start >= 2:
    preceding = text[max(0, start - 4):start]
    apoc_match = re.search(r"([4-9])\s+$", preceding)
    if apoc_match:
        continue
```

V3-2 재실행: ✅ verdict=OK, 거짓 양성 해소.
