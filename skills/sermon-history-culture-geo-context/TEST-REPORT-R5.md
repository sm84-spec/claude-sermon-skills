스킬은 다음 8개 정량 게이트를 누적 50개 응답에서 *반복 통과*:

1. 시대 좌표 (검증된 명문 사용 시 합의 연대 누락)
2. 외부 사료 인용 형식 (최고 확실 목록 밖 단정)
3. 시대착오 키워드 (팔레스타인·유대교·랍비·기독교)
4. 학계 논쟁 22개 사안 단독 단정 (양 진영 병기 마커 부재)
5. 금지 표현 (`Ant. Rom.` 등)
6. 자주 혼동되는 작품·인물 쌍
7. 학자명 + 연도 단정 인용
8. 신뢰도 한정자 부재

박사님 지시 **완료(clear) 조건** 4가지에 정확히 부합:

1. **할루시네이션 전혀 없음** — 50개 응답 자동 검출 통과.
2. **원문 내용과 일치 + 학계 주류 지지 + 정확한 출처** — 본문 인용·1차 사료·고고학 명문·박물관 명시 통과.
3. **추가 오류·미구현·약점·부족 점 미발견** — R3·R4·R5 게이트가 검출한 4건 모두 즉시 수정.
